## 🚀 Быстрый старт

### 1. Установка (30 сек)
```bash
pip install -r requirements.txt
```

### 2. Конфигурация (1 мин)
Создайте `.env`:
```env
TELEGRAM_TOKEN=your_token
OPENROUTER_API_KEY=your_key
YOOKASSA_SHOP_ID=your_id
YOOKASSA_SECRET_KEY=your_secret
OMNIGUARD_SECRET=your_secret
```

### 3. Запуск (10 сек)
```bash
python bot.py
python webhook.py  # В отдельном окне
```

Бот готов! 🎉
