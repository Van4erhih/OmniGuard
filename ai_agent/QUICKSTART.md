# 🚀 OmniGuard - Быстрый старт

## За 5 минут до рабочего бота

### Шаг 1: Клонирование и установка (1 мин)

```bash
# Клонировать проект
git clone https://github.com/Van4erhih/OmniGuard.git
cd OmniGuard

# Установить зависимости
pip install -r requirements.txt
```

### Шаг 2: Получить ключи (2 мин)

#### Telegram Bot Token
1. Откройте Telegram
2. Напишите @BotFather
3. Создайте бота (/newbot)
4. Скопируйте токен

#### OpenRouter API Key
1. Зайдите https://openrouter.ai
2. Зарегистрируйтесь/авторизуйтесь
3. Получите API ключ

#### Yookassa (пока пропустим регистрацию)
- Для тестирования оставьте временные значения в .env

### Шаг 3: Создать .env файл (1 мин)

Создайте файл `.env` в корневой папке проекта:

```env
TELEGRAM_TOKEN=ваш_токен_от_botfather
OPENROUTER_API_KEY=ваш_ключ_openrouter
YOOKASSA_SHOP_ID=test_shop_id
YOOKASSA_SECRET_KEY=test_secret_key
OMNIGUARD_SECRET=omniguard_secret
```

### Шаг 4: Запустить бот (1 мин)

```bash
python bot.py
```

Вы должны увидеть:
```
🛡️ OmniGuard запущен и работает...
```

## ✅ Готово!

Бот работает. Найдите его в Telegram и напишите `/start`

### Зарегистрироваться в Yookassa (ВАЖНО!)

1. https://yookassa.ru
2. Создайте личный кабинет
3. Создайте магазин
4. Скопируйте Shop ID и Secret Key
5. Обновите .env
6. Перезагрузите бот

### Развернуть вебхук сервер

В отдельном терминале:
```bash
python webhook.py
```

Это нужно для обработки платежей от Yookassa.

## ❓ Часто встречаемые проблемы

### "ModuleNotFoundError: No module named 'aiogram'"
```bash
pip install -r requirements.txt
```

### "Failed to connect to Telegram"
- Проверьте интернет соединение
- Проверьте TELEGRAM_TOKEN в .env

### "YOOKASSA_SHOP_ID not found"
- Обновите .env с правильными значениями
- Перезагрузите бот

### Большой файл не отправляется
- Бот автоматически разделит его на части
- Следуйте инструкциям для сборки
