## 💾 База данных

Структура SQLite (omniuard.db):

TABLE users_extended
  - user_id (PRIMARY KEY)
  - created_at, updated_at

TABLE subscriptions
  - id (PRIMARY KEY)
  - user_id (UNIQUE)
  - plan_key, start_date, end_date
  - is_active

TABLE payments
  - id (PRIMARY KEY)
  - user_id, yookassa_payment_id
  - plan_key, amount, currency
  - status, created_at, updated_at
  - metadata

TABLE gift_codes
  - id (PRIMARY KEY)
  - code (UNIQUE)
  - plan_key, created_by, used_by
  - created_at, used_at, is_used

## 📝 Инструкции для старта

1. Установить зависимости:
   pip install -r requirements.txt

2. Создать .env с ключами:
   TELEGRAM_TOKEN=...
   OPENROUTER_API_KEY=...
   YOOKASSA_SHOP_ID=...
   YOOKASSA_SECRET_KEY=...

3. Запустить бот:
   python bot.py

4. Запустить вебхук сервер (в отдельном окне):
   python webhook.py

5. Написать /start боту в Telegram
