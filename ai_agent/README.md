"# 🛡️ OmniGuard - Telegram AI Assistant

Интеллектуальный AI-ассистент для Telegram с поддержкой платных функций, парсинга маркетплейсов и генерации Excel отчётов.

## ✨ Возможности

### 🆓 Бесплатные функции
- **AI Ассистент** - Ответы на вопросы, анализ информации
- **Калькулятор** - Математические вычисления
- **Конвертер единиц** - Преобразование единиц измерения

### 💳 Платные функции (требуют подписку)
- **Парсер маркетплейсов** - Поиск товаров на Озоне, Валберис, Яндекс Маркете
- **Сравнение цен** - Анализ цен одного товара на разных маркетплейсах
- **Excel генератор** - Создание отчётов о товарах, ценах, задолженности

## 🚀 Установка

### Требования
- Python 3.9+
- pip

### Шаги установки

1. **Клонировать репозиторий**
```bash
git clone https://github.com/Van4erhih/OmniGuard.git
cd OmniGuard
```

2. **Установить зависимости**
```bash
pip install -r requirements.txt
```

3. **Создать .env файл** в корневой директории
```env
TELEGRAM_TOKEN=your_telegram_bot_token
OPENROUTER_API_KEY=your_openrouter_api_key
YOOKASSA_SHOP_ID=your_shop_id
YOOKASSA_SECRET_KEY=your_secret_key
OMNIGUARD_SECRET=your_secret_key
```

### Получение ключей

#### Telegram Bot Token
1. Напишите @BotFather в Telegram
2. Создайте нового бота
3. Скопируйте токен

#### OpenRouter API Key
1. Зарегистрируйтесь на https://openrouter.ai
2. Получите API ключ в личном кабинете

#### Yookassa (для платежей)
1. Зарегистрируйтесь на https://yookassa.ru
2. Создайте магазин
3. Получите Shop ID и Secret Key

## 📦 Обработка больших файлов

Если Excel файл превышает лимит Telegram (50 MB), бот автоматически:
1. Разбивает файл на части
2. Отправляет каждую часть отдельно
3. Предоставляет инструкции по сборке

## 🏗️ Архитектура

### Система подписок
- SQLite база для управления подписками
- Автоматическое истечение после окончания периода
- Подарочные коды для промо и подарков

### Платежная система
- Интеграция с Yookassa
- Вебхуки для подтверждения платежей
- Автоматическая активация после оплаты

### AI Агент
- OpenRouter API для работы с LLM
- История чатов с шифрованием
- Поддержка расширяемых инструментов

### Парсеры
- BeautifulSoup4 для парсинга HTML
- Поддержка Озона, Валберис, Яндекс Маркета
- Защита от блокировки (User-Agent headers)

## 🔒 Безопасность

- Пароли и ключи хранятся в .env
- История чатов зашифрована
- Подписи вебхуков проверяются
- Rate limiting для парсеров

## 📝 Разработка

### Добавление новых инструментов

1. Создайте класс в `tools/`
```python
from tools.base import BaseTool

class MyTool(BaseTool):
    name = "my_tool"
    description = "My tool description"
    
    def run(self, **kwargs):
        # Реализация
        pass
```

2. Добавьте в `tools/registry.py`
```python
from .my_tool import MyTool
TOOLS["my_tool"] = MyTool()
```

## 🐛 Известные ограничения

- Парсеры могут блокироваться маркетплейсами (требуется Selenium для сложных случаев)
- Лимит на размер файлов в Telegram (документы до 2GB, но рекомендуется <= 50MB)
- OpenRouter имеет rate limits
